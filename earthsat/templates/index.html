• GET - Show a list of all satellites in the database.
    api/satellites

  • GET - Show a list of all operators in the database
    api/operators

  • GET - api/satellite/[satellite id] - show the details associated with the satellite corresponding to the given ID. Can Also be used to update and delete the satellite.
    api/satellites/1634

  • GET - api/operator/[operator id] - show the details associated with the operator corresponding to the given ID. Can Also be used to update and delete the operator.
    api/operator/230

  • POST - Add a new satellite to the database. The URL below link to a page where a form allow us to make a post request. Validation is handled automatically through the DRF serializer. See below for a valid JSON to test the endpoint.
    api/satellite/add

  • GET - api/satellite/query?[param1]=[value]&[param2]=[value]&... - Filter the satellites list based on the query parameters. Accept queries of any complexity.
    api/satellites/query?country_un_registery=USA - Show all satellites registered in the USA
    api/satellites/query?country_un_registery=USA&country_un_registery=Finland - Show all satellites registered in the USA OR Finland
    api/satellites/query?country_un_registery=USA&orbit_class=GEO - Show all satellites registered in the USA with a geostationnary orbit
